HTML Help file authoring notes
------------------------------
* Screenshots are based on the Archi application being sized to 1011x763 pixels
* No plug-ins are installed!
* -Dshowheap is off!
* Images should be no wider than 700 pixels
* Set the width and height attributes of all images in <img> tags (half the values for 2x hi-res images)
* Have one only <p>&nbsp;</p> at the end of each HTML file


Edit the source HTML files in the com.archimatetool.help/help folder
--------------------------------------------------------------------
* Ensure that "contexts.xml" is up to date
* Ensure that "toc.xml" is up to date and corresponds to com.archimatetool.build/html_contents.list
